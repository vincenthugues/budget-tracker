# Budget Tracker

## Description

Budget tracking app written with Nest and React, with json-based YNAB data import and simple transactions/categories/etc. list views

## Dependencies

mongodb

## Run locally

- `yarn server:db`
- `yarn server:start:dev`
- `yarn client:start:dev`
